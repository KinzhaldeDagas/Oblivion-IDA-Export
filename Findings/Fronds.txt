Request addressed: "Find all Frond Hooks and places of interest in relation to SpeedTree Frond/Fronds"

Scope and method
- Searched for explicit hook-style functions and checked whether any are Frond/SpeedTree-Frond specific.
- Enumerated Frond/Fronds/SpeedTreeFrond string anchors from metadata strings and mapped their xrefs to concrete function locations.
- Collected directly related constructor/destructor/type-registration points for SpeedTreeFrondShader.

Commands used
- rg -n -i "speedtree|frond|fronds" .
- Python scan of functions_index_by_name.json for names containing "hook"
- Python scan of data/named_items/_index.json + xrefs_to.json for entries containing "frond"
- rg -n -i "Frond|CSpeedTreeRT::SetFrond|DeleteFrond|CFrondEngine|BSSM_FRONDS|speedtree\\frond|STFROND|SpeedTreeFrondShader" functions metadata/strings.json data/named_items/_index.json
